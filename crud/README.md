# crud

## usage

```sh
# write about db config in application.yaml
cp src/main/resources/config/application.yaml.sample src/main/resources/config/application.yaml

# run spring-bott
mvn clean spring-boot:run

# show hello world
open http://localhost:8080
```

- Java: 1.8.0
- Apache Maven: 3.5.3
